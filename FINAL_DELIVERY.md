## 🚀 HOW TO LAUNCH THE DEMO

### STEP 1: Start Backend

```bash
cd server
npm start
# Server will run on http://localhost:5000
```

### STEP 2: Start Frontend (in new terminal)

```bash
cd client
npm run dev
# Frontend will run on http://localhost:5173
```

### STEP 3: Open in Browser

```
http://localhost:5173
```
